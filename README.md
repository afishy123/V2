Purpose: 

 Characterizing whether sequence to function models can spot enhancer hijacking. Next thing to do is test more phenomonon. 

 
How: 

 For a set of known fusion cases, we build the 1 Mb WT sequence and the rearranged version, run both through each the model, and see if the predicted RNA seq or Chip seq signal on the oncogene goes up the way it should, using shuffled sequence controls and distance sweeps to make sure the signal is real.
